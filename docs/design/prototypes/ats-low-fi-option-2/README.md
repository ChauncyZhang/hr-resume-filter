# 招聘协同平台低保真原型（方案 2）

这是 UX-01 的可点击桌面端原型，视觉基准为 `reference-option-2.png`。

## 覆盖范围

- 工作台与职位切换
- 候选人六阶段看板与列表视图
- 简历批量导入模拟
- 候选人详情、AI 评估与阶段推进
- 重复候选人合并处理
- 待办事项、面试日历与移动端响应式布局
- 职位列表搜索与筛选（JOB-01）
- 新建/编辑职位、JD 条件提取与发布检查（JOB-02）
- 职位漏斗、候选人、协作动态和职位设置（JOB-03）
- 分步导入、来源确认和文件校验（IMP-01）
- 逐文件筛选进度、规则/LLM 结果和失败重试（SCR-01）
- 筛选结果搜索、状态筛选、批量推进与最近任务恢复
- 跨职位候选人搜索、筛选、比较与批量处理（CAN-01）
- 候选人完整档案、筛选证据、面试反馈与活动时间线（CAN-02）
- 受控状态推进、淘汰原因校验与并发更新冲突提示

## 本地运行

```bash
npm install
npm run dev
```

原型使用模拟数据，不连接生产服务，也不包含真实候选人个人信息。
